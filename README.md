Ashin Krishna | Portfolio & Backend Service
A modern, responsive personal portfolio website featuring a "glassmorphism" UI and a functional contact form powered by a Node.js/Express backend with PostgreSQL integration.

🚀 Live Demo
Frontend: https://ashin-krish.github.io/project-portfolio/

Backend: https://project-portfolio-7ovh.onrender.com

🛠️ Tech Stack
Frontend
HTML5 & CSS3: Semantic layout and custom Glassmorphism styling.

JavaScript (ES6+): Asynchronous API handling with fetch.

FontAwesome: For professional iconography.

Backend
Node.js & Express.js: RESTful API development.

PostgreSQL: Relational database for storing user messages.

CORS: Configured for cross-origin resource sharing.

node-postgres (pg): Interface for PostgreSQL.

✨ Features
Responsive Design: Fully optimized for mobile, tablet, and desktop views.

Glassmorphism UI: Modern frosted-glass aesthetic using backdrop-filter.

Functional Contact Form: Validates user input and sends data to a live database.

Scalable Backend: Ready-to-use Express server with health check endpoints and PostgreSQL table auto-initialization.
